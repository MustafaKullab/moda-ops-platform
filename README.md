## 🔧 Setup & Installation

```bash
# Clone the repository
git clone https://github.com/MustafaKullab/moda-ops-platform.git

# Navigate into the project folder
cd moda-ops-platform

# Install dependencies
npm install

# Run development server
npm run dev
```
